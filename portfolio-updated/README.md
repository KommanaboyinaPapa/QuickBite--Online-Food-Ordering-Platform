### Option 1: GitHub Pages (Recommended)

1. **Create a GitHub Repository**
   - Go to [GitHub](https://github.com) and create a new repository
   - Name it: `portfolio2` (or any name you prefer)
   - Make it public

2. **Upload Your Files**
   ```bash
   cd portfolio-updated
   git init
   git add .
   git commit -m "Updated portfolio with web development experience"
   git branch -M main
   git remote add origin https://github.com/kommanaboyinapapa/portfolio2.git
   git push -u origin main
   ```

3. **Enable GitHub Pages**
   - Go to your repository settings
   - Navigate to "Pages" section
   - Under "Source", select "main" branch
   - Click Save
   - Your site will be live at: `https://kommanaboyinapapa.github.io/portfolio2/`

### Option 2: Update Existing Repository

If you already have the portfolio2 repository:

1. **Clone Your Existing Repository**
   ```bash
   git clone https://github.com/kommanaboyinapapa/portfolio2.git
   cd portfolio2
   ```

2. **Replace Files**
   - Copy `index.html`, `style.css`, and `script.js` to the repository
   - Keep your existing `profile.jpg` and `KOMMANABOYINA-PAPA-resume.pdf`

3. **Commit and Push**
   ```bash
   git add .
   git commit -m "Update portfolio with web development experience and QuickBite project"
   git push origin main
   ```

4. **Wait for Deployment**
   - GitHub Pages will automatically rebuild your site
   - Check your site at: `https://kommanaboyinapapa.github.io/portfolio2/`
